Sobre & Novidades

O que é o HydroFetch?

HydroFetch é um utilitário de linha de comando escrito em Shell Script, criado para exibir informações do sistema de forma elegante e organizada diretamente no terminal. Inspirado por ferramentas como Neofetch, ele usa o Figlet para gerar a arte ASCII do nome da sua distribuição e ícones Nerd Font para uma apresentação visual limpa.

Recursos Principais

  • Duas Visualizações: Uma visualização padrão, bonita e concisa, e uma completa com o parâmetro --all para quem precisa de todos os detalhes.
  • Fontes Customizadas: Personalize a arte ASCII do seu sistema operacional! Basta adicionar um arquivo de fonte Figlet (Custom.flf) na pasta ~/.hydrofetch.
  • Instalação de Dependências: O script verifica e instala automaticamente o Figlet, caso ele não esteja presente no seu sistema.
  • Easter Egg: Descubra um segredo com o parâmetro --tux.

Informações Exibidas

Visualização Padrão

  • Arte ASCII do SO
  • Usuário
  • Host
  • Sistema Operacional
  • Kernel
  • Ambiente de Desktop (DE)
  • Uso de RAM
  • Cores do Terminal

Visualização Completa (--all)

  • Hostname e Distribuição
  • Kernel, Arquitetura e Tipo de SO
  • Modelo e núcleos da CPU
  • Memória RAM Total
  • Uso de Disco
  • Usuário Logado e diretório Home
  • Uptime do Sistema
  • Endereço IP e Interface de Rede

Como Instalar o HydroFetch

1

Copie o Script

Primeiro, pegue o script de instalação diretamente do nosso repositório. Clique aqui para ver o script.

2

Salve o Arquivo

Salve o conteúdo copiado em um arquivo chamado .hydrofetch.sh na sua pasta home (~/).

3

Dê Permissão de Execução

Abra seu terminal e execute o seguinte comando para tornar o script executável: 

chmod +x ~/.hydrofetch.sh
4

Execute

Agora você pode rodar o HydroFetch com:

~/.hydrofetch.sh

O script se encarregará de baixar as dependências necessárias, como o FIGlet.

Requisitos

Um sistema operacional baseado em Linux (Debian, Arch, Fedora, Alpine, etc.).

Uma Nerd Font instalada e configurada no seu emulador de terminal.
